# Square-Up Calculator

A simple field calculator for squaring up building, slab, and layout dimensions.

## Live Use

Open the hosted site in Safari on iPhone, then use `Share > Add to Home Screen` to install it like an app.

## Files

- `index.html` redirects the site root to the calculator.
- `square-up-calculator.html` contains the app.
- `square-up-calculator.webmanifest` and `square-up-sw.js` support install and offline use.
